UPS Shipping method for WooCommerce site. Easy installation and very light for WooCommerce sites.

== Description ==

Get UPS shipping rate for your website. This simple,light plugin just needs your UPS credentials as a setup and you are done. Your customers will start getting UPS shipping rates straight way.

From admin settings you can select shipping services (UPS services), that your shop supports. Users will get rates only for those shipping services.

This plugin is not as complex as other UPS plugins available in the web. This is a light version. It calculates cutomer's cart weight and send it to UPS API and get live UPS shipping rates for your customer.

For any further query you can contact us here in the support forum or our [website](http://www.mingocommerce.com/contact-us/). Our technical supports guys will guide you through any issues.

== Installation ==

Manual

1. Upload the plugin folder to the '/wp-content/plugins/' directory .

2. Activate the plugin through the 'Plugins' menu in WordPress.

Automatic

1. Go to plugin list from 'dashboard'. Search our plugin and click to install

2. Activate it.
